# Sistema de Gestión Escolar (SGE) - Community Edition
> *Administración académica robusta diseñada para entornos sin conexión (Offline-First).*



Las escuelas en zonas rurales o con conectividad intermitente sufren con sistemas web que requieren internet constante. La gestión en papel (planillas, listas) es lenta, insegura y propensa a pérdidas, afectando la eficiencia administrativa y la experiencia de los estudiantes.

## 💡 La Solución
Una plataforma de escritorio **Offline-First** que lleva la experiencia de usuario de una web moderna a una aplicación local segura. Permite gestionar todo el ciclo escolar (inscripción, evaluación, reportes) sin depender de la nube.

### Características Clave
- ✅ **Arquitectura 100% Offline**: Base de datos encriptada localmente. Funciona siempre, haya luz/internet o no.
- ✅ **Conectividad Local (Híbrida)**: Crea su propia red local para que celulares/tablets se conecten vía QR para toma de asistencia.
- ✅ **Wizard de Inscripción**: Flujo paso a paso con validación de duplicados y generación automática de expedientes.
- ✅ **Centro de Impresión**: Generación nativa de constancias y reportes oficiales en formato A4 exacto.

### Tecnologías Utilizadas
| Categoría | Tecnologías |
|-----------|-------------|
| **Core** | Electron + React (Vite) |
| **UI/UX** | Tailwind CSS + Framer Motion (Glassmorphism) |
| **Persistencia** | Base de datos NoSQL Local (Encriptada) |
| **Seguridad** | Hashing de credenciales y IPC Bridge seguro |

## 🚀 Impacto
- **Autonomía Total**: Eliminación de la dependencia de proveedores de internet para operar.
- **Eficiencia**: Reducción drástica del tiempo de inscripción por alumno.
- **Seguridad**: Datos sensibles protegidos físicamente en la institución.
