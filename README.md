## 📖 About the Project
A powerful desktop application designed to provide modern academic administration in areas with limited or no internet connectivity. This system brings the efficiency of a cloud web application to a secure, locally-hosted environment.

## ✨ Key Features
- 🔒 **Local Encryption**: Fully functional offline architecture with robust local data encryption.\n- 📡 **Hybrid Networking**: Local network capabilities for mobile access within the facility.\n- 🖨️ **Native Printing**: Automated enrollment processes and native document generation.
